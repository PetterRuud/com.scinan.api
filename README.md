# Scinan
Support wifi climate controllers using the Scinan API https://api.scinan.com.

## Supported Devices

#### Saswell
 - SAS920FHL-7W-WIFI

#### Prosmart
 - PST50W

## Supported features
 - [x] Turn On/Off
 - [x] Set Away/Home
 - [x] Set target temperature
 - [x] View mesured tempature
 - [x] Set mode (Comfort, Day or Night or Auto)
 - [ ] Program

## Supported Languages:
 - English
